# Hammerspoon キーバインド

Hammerspoonのウィンドウ管理とアプリケーション起動のキーバインド一覧。

## 設定ファイル

- メイン設定: [.hammerspoon/init.lua](../.hammerspoon/init.lua)
- Ctrlダブルタップ検知: [.hammerspoon/ctrlDoublePress.lua](../.hammerspoon/ctrlDoublePress.lua)

## 設定管理

| キーバインド | 機能 |
|-------------|------|
| `⌘ + ⌥ + ⌃ + R` | Hammerspoon設定をリロード |

## ウィンドウ配置

### 半分サイズ配置

| キーバインド | 機能 |
|-------------|------|
| `⌘ + ⌥ + ←` | ウィンドウを画面左半分に配置 |
| `⌘ + ⌥ + →` | ウィンドウを画面右半分に配置 |
| `⌘ + ⌥ + ↑` | ウィンドウを画面上半分に配置 |
| `⌘ + ⌥ + ↓` | ウィンドウを画面下半分に配置 |

### 4分の1サイズ配置

| キーバインド | 機能 |
|-------------|------|
| `⌘ + ⌥ + ⌃ + ←` | ウィンドウを左上4分の1に配置 |
| `⌘ + ⌥ + ⌃ + ↑` | ウィンドウを右上4分の1に配置 |
| `⌘ + ⌥ + ⌃ + ↓` | ウィンドウを左下4分の1に配置 |
| `⌘ + ⌥ + ⌃ + →` | ウィンドウを右下4分の1に配置 |

### その他の配置

| キーバインド | 機能 |
|-------------|------|
| `⌘ + ⌥ + F` | ウィンドウを最大化 |
| `⌘ + ⌥ + C` | ウィンドウを中央に配置（画面の60%サイズ） |

### マルチディスプレイ

| キーバインド | 機能 |
|-------------|------|
| `⌘ + ⌥ + N` | ウィンドウを次のディスプレイへ移動 |

## アプリケーション起動

| キーバインド | 機能 |
|-------------|------|
| `⌃` を2回タップ（0.5秒以内） | WezTermを表示/非表示切り替え |

## 記号の説明

- `⌘`: Command (Cmd)
- `⌥`: Option (Alt)
- `⌃`: Control (Ctrl)
- `⇧`: Shift

## カスタマイズ

設定をカスタマイズする場合は [.hammerspoon/init.lua](../.hammerspoon/init.lua) を編集してください。
編集後は `⌘ + ⌥ + ⌃ + R` で設定をリロードできます。
